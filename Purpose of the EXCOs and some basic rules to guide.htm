Purpose of the EXCOs and some basic rules to guide:

The EXCOs go on a monthly basis to the board members of ECAG. They provide an overview of open recommendations assigned to each board member based on their areas of responsibility.

Some rules to be aware of:

         If recommendations come from an S3 finding, then they get colored orange.

         If a recommendation is overdue then it must be colored pink.

         The order of recommendations must be determined by the severity. S3, then S2, then S1. In case of overdue recommendations, the pink overdue goes first in line and then the rest are ordered by severity.

         When you filter by date to see if there are overdue recommendations always check first with SAP SAM. It may also be the case that the deadline has been extended. If that is the case, dont forget to update on our reporting columns Current Deadline and No. of Deadline Shifts.

         If the recommendation is overdue on the system as well, then make sure to check with the Overdue/Close to Overdue Recommendations and Prolongations email that Group Audit sends on Fridays. There you will find a link to the file where auditors input details regarding the overdue. Should there not be enough information, you can directly contact the auditor in charge.

You will find multiple sheets on the EXCO reporting. The main one is called ECAG open Recos. Remember to save always on this page, this is the first thing that everyone who receives the report should see.

How to prep the EXCOs

Trigger: Group Audit sends the monthly reporting of open Findings and Recommendations. This allows us to delete recommendations that have been completed and add new ones from all the audits that have come out in time.

Remember: We only report open recommendations.

The file that we receive from Group Audit has two sheets: one for Findings and one for Recommendations. Always use Recommendations. One Finding has at least one recommendation but in many cases, there can be multiple recommendations for 1 Finding.

We usually use VLOOKUP to check for new recommendations and discard already completed ones. Use Recommendation ID as a reference.

Feel free to use any type of formula you see fit, what also works is XLOOKUP or the INDEX/MATCH combination.

What the arguments look like if you use VLOOKUP.

 

This step shows you what recommendations have been completed. If the recommendations are not found in the Group Audit file, it means that they have been closed.

Repeat the same process on the file given by Group Audit. Like this you will see which recommendations are new. Copy and paste the new recommendations into the ECAG open Recos sheet.

Remember that the columns are not in the same order on both files.

Next classify to the pertaining Board Member and Category.

Filter for each ECAG Board Member as shown in the screenshot below.

There is only one ECAG Board Member for each recommendation. If you see two ECAG Board Members, then a mistake might have happened.

Remember to check that you have the correct Board Member. Several Board Members of ECAG are also Board Members of Eurex Repo or Eurex Frankfurt. Just seeing the name does not guarantee you automatically have the correct member; pay attention to the brackets to see which entity it is. We are only interested in Eurex Clearing AG

New recommendations must be assigned to the sheets of the respective Board Members. You also must remove the completed recommendations from the Board Members sheet. You can use the same approach as the one of the main sheet.

 

For statistic purposes we have to also separate the Category Column.

Example: If you see under Category Control Des. + Control Op. Eff., this means you will put Yes under Columns Control Design Effectiveness and Control Operating Effectiveness and No under Control Environment.
